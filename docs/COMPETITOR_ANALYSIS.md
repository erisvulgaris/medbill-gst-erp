# Competitor Analysis

> **Audit date:** 2026-06-30
> **Method:** Public documentation review, feature comparison, pricing pages
> **Scope:** 8 Indian GST billing/ERP products
> **Note:** Based on publicly available information as of audit date. Feature sets change; verify before relying.

## Executive Summary

The Indian GST billing market is crowded with 8+ established players. MedBill's differentiation opportunity is in **performance, UX, and developer maintainability** — not feature parity (TallyPrime and Zoho Books have decade-long feature leads).

| Product | Target | Strength | Weakness | MedBill Opportunity |
|---------|--------|----------|----------|---------------------|
| myBillBook | SMB retail | Easy onboarding, mobile | Limited customization | Better UX + speed |
| Vyapar | SMB retail/wholesale | Offline desktop, barcode | Dated UI | Modern web UX |
| Busy Accounting | Mid-market | Accounting depth | Complex, Windows-only | Simpler + web |
| TallyPrime | SMB+ enterprise | Industry standard, depth | Steep learning curve | 5-min onboarding |
| Marg ERP | Retail chains | Inventory depth | Clunky UX | Polish |
| Zoho Books | Service/SMB | Integration ecosystem | Not inventory-heavy | Inventory + speed |
| Zoho Inventory | Inventory-heavy | Supply chain | No accounting | Unified billing+inventory |
| Swipe | Mobile-first SMB | Fast invoicing | Shallow features | Depth + speed |

## MedBill's Strategic Position

MedBill targets the gap between **Swipe (too simple)** and **Vyapar/Tally (too complex/dated)**:
- **Faster than Tally** onboarding (2min vs 1hr+)
- **More modern** than Vyapar/Marg (web-first, not Windows)
- **Deeper** than Swipe/myBillBook (full inventory + accounting + GST)
- **More affordable** than Zoho Books (free tier)

**Key differentiators to build:**
1. **Sub-5-second invoice creation** (competitors average 15-30s)
2. **120 FPS web app** (competitors are desktop or slow web)
3. **Offline-first PWA** (cloud competitors lack this)
4. **Modern design system** (emerald, Linear/Stripe quality)
